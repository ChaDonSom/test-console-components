# Test Console Components

This repository is a proof-of-concept to create a Vue component library.

## Project setup

```sh
npm install
```

## Testing the components

```sh
npm run dev
```

### Your Vue project

```vue
<script setup>
import { TestButton } from "test-console-components"
</script>

<template>
  <div id="app">
    <TestButton />
  </div>
</template>
```
